STYLE: ASDF Movie-style animation frame. Every frame looks drawn by the same hand with the same tools — medium-weight black marker on flat white paper. Clean, minimal, instantly readable.

FIGURES: Pure stick figures. Head = simple circle. Body = one vertical line. Arms = two angled lines from shoulder. Legs = two lines from hip with small oval feet. Eyes = two dots. Mouth = curve up (happy), curve down (sad), circle (shocked), flat line (neutral). Stick figures are ALWAYS filled white inside — their interior is never colored. Black outline only, white fill.

CHARACTER CONSISTENCY: If a character reference sheet is provided, match that design EXACTLY in every frame — same proportions, same identifying features, same line weight. Identical across all appearances.

BACKGROUND: Always solid flat white (#FFFFFF). The background area is never colored, never filled. No textures, no paper grain, no gradients, no atmospheric effects.

COLOR — USE FREELY BUT WITH DISCIPLINE:
Colors appear on objects, props, and environment elements only — not on backgrounds, not inside stick figures.
Examples: grass patch = flat green, a car = flat red, a house = flat yellow walls + blue roof, fire = orange, sky element = light blue shape, money = flat green, a phone screen = any color.
Pick colors that feel semantically right for the object. Use a consistent palette across the whole video — if a car is red in frame 3, it is red in frame 10. Introduce a color only when there is a clear visual reason. Maximum 3–4 flat colors per frame besides black and white. No gradients, no shading, no blending, no transparency effects.

COMPOSITION: Wide 16:9. Subject centered. Generous empty white space around it. One clear action or emotion per frame. Characters interacting: face each other with space between.

PROPS / ENVIRONMENT: Suggested with 1–3 minimal geometric shapes. A floor = horizontal line. A building = rectangle with a few lines. A tree = circle on a stick. A road = two parallel lines. When a scene continues in the same location, reuse the exact same props and their colors.

SPEECH / TEXT: Only when the scene description specifically calls for it. Short, hand-lettered style, inside a simple oval speech bubble.

DO NOT: No realistic anatomy, no gradients, no shadows, no 3D, no photorealism, no anime/manga, no complex backgrounds, no watermarks, no extra limbs, no broken anatomy, no colored stick figure interiors, no colored backgrounds.
